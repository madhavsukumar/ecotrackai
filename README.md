EcoTrack AI is a web-based prototype that helps users estimate their carbon footprint based on daily spending categories.  
The goal of the project is to create awareness about the environmental impact of everyday financial decisions in a simple and user-friendly way.

This project was developed during a 36-hour online hackathon by first-year Computer Science students.

## ğŸ’¡ Proposed Solution
EcoTrack AI allows users to:

- Receive simple suggestions to reduce environmental impact

The focus is on **simplicity, accessibility, and awareness**.
